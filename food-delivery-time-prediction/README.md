# Food Delivery Time Prediction

## Project Overview
This project is a Machine Learning model that predicts food delivery time based on factors such as distance, traffic conditions, weather, restaurant preparation time, and delivery partner availability.

## Objectives
- Analyze delivery data
- Perform data preprocessing and cleaning
- Conduct exploratory data analysis (EDA)
- Build a regression model to predict delivery time
- Visualize insights using charts

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Structure
- notebook/ → Jupyter notebook with analysis & model
- data/ → Dataset files
- outputs/ → Generated charts and results

## Results
The model predicts delivery time with reasonable accuracy based on real-world delivery conditions.
