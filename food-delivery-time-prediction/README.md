# 🍔 Food Delivery Time Prediction

## 📌 Project Overview
This project is a Machine Learning model that predicts food delivery time based on real-world factors such as traffic conditions, weather, order time, restaurant preparation time, and delivery partner details. The goal is to improve delivery time estimation, customer satisfaction, and operational efficiency.

## 🎯 Objectives
- Analyze food delivery dataset  
- Perform data cleaning and preprocessing  
- Conduct Exploratory Data Analysis (EDA)  
- Identify key factors affecting delivery time  
- Build a regression model to predict delivery time  
- Visualize insights using graphs and charts  

## 🛠️ Technologies Used

- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn

## 🔍 Key Insights
- Traffic is the strongest factor affecting delivery time  
- Festival days significantly increase delivery delays  
- Evening hours have higher delivery time due to peak demand  
- Higher-rated delivery personnel deliver faster  
- Weather conditions like fog increase delivery time  

## 🤖 Machine Learning Model
- Type: Regression Model  
- Goal: Predict delivery time (in minutes)  
- Techniques used: Data Cleaning, Feature Engineering, One-Hot Encoding, Outlier Handling (IQR Method)

## 📊 Results
- Final dataset: 43,607 records  
- Strong impact of traffic and festival conditions on delivery time  
- Model captures real-world delivery patterns effectively
